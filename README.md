# Box

This is a script to generate ASCII-style boxes from words.

## Example
```
$ ./box.py AESTHETIC
        C I T E H T S E A
      / I             / E
    /   T           /   S
  /     E         /     T
A E S T H E T I C       H
E       T       I       E
S       S       T       T
T       E       E       I
H       A E S T H E T I C
E     /         T     /
T   /           S   /
I /             E /
C I T E H T S E A
```
